# Portfolio

The live site is the Round 5 polished self-contained portfolio prototype.

## Live entry

The Next.js home page redirects to:

```text
/portfolio/round-5/
```

The standalone prototype files live in:

```text
public/round-5/
```

## Commands

```bash
npm install
npm run dev
npm run build
```

`npm run build` creates the static export in `out/` for GitHub Pages.
